## ✨ Features
- In-app update checker
- Android Auto support
- Youtube music playlist import
- Synced lyrics from lrclib.net
- Song download
- Simple sleep timer
- Opening Youtube music links
- Search Youtube music for songs, albums, artists, playlists and videos
